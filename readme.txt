== Description ==
Plugin automatically publishes posts from the wordpress blog on a service Blog.ru. The plugin sends the letter to service Blog.ru.

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
Q: How do I set a publish my posts on Blog.ru?<br> 
R: Pass in the admin menu and set user login and code word by account Blog.ru.
